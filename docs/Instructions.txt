WEBSITE CÂU LẠC BỘ AI AGENT - HƯỚNG DẪN TÀI LIỆU

Điều hướng nhanh:
├── 01-project-requirements.md    → Tính năng, trang web, tổng quan chức năng
├── 02-authentication-system.md  → Hệ thống đăng nhập, database schema, cài đặt admin
├── 03-design-system.md          → Màu sắc, font chữ, UI components, styling
└── 04-project-structure.md      → Tổ chức code, thiết lập development

Thứ tự đọc:
1. Bắt đầu với 01-project-requirements.md để hiểu chúng ta sẽ xây dựng gì
2. Xem 02-authentication-system.md để thiết lập database và quản lý user
3. Tham khảo 03-design-system.md cho approach UI/UX styling
4. Xem 04-project-structure.md về tổ chức code và development cục bộ

Ghi chú triển khai:
- SQL schemas có thể copy-paste trực tiếp vào Supabase SQL Editor
- Dùng "npm run dev" để test development cục bộ
- Test production builds bằng "npm run build && npm run start" trước khi deploy
- Deploy lên Vercel thông qua tích hợp GitHub

Mỗi file độc lập - nhảy đến phần cụ thể khi cần trong quá trình development.
